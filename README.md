# blog_project

1 Nodemon install

```
yarn global add nodemon
or
npm i -g nodemon
```

1.1 Nodemon run

```

nodemon

```

2 Typescript install

```

yarn global add typescript

```

3 tsc install

```

yarn tsc --init

```

4 ts-node install

```

yarn add ts-node --dev

```

5 Node types

```

yarn add @types/node

```

6 START SERVER

```

yarn start

```

7 yarn express

```

yarn add express

```

8 express types

```

yarn add @types/express

```

9 yarn install

```

npm i yarn

```

10 TypeORM install

```

yarn add typeorm

```

11 mySQL install

```

yarn add mysql

```
